# smartthings_drivers

This repository stores SmartThings Edge drivers. It includes drivers for devices I developed and Tuya-based drivers.

## Install

- SmartThings channel: https://bestow-regional.api.smartthings.com/invite/q0jmrwAZeaM1

## Contents

- Drivers for devices I developed
- Edge drivers for Tuya devices

## wp30-eu-plug

- Target device: LELLKI WP30-EU (power monitoring)
- Fingerprints: TS011F / _TZ3000_c7nc9w3c, _TZ3210_c7nc9w3c
- Driver path: `wp30-eu-plug`
